Basic Web Concepts


Overview

The world wide web is a collection of html docs and other types of files. HTML is the main document typee and glues everything together. It is made up of text, markup, references to other docs and links to other pages.

HTML

The language of the web. There are syntax and rules. BRowser turns HTML tags to elemetnts into a tree like structure (the DOM). Each tag in html has a name and describes what element is created. The content goes in between the tags.

  • HTML tags <> and </> (has opening and closing tags)
  • Examples: bold (b), italics (em)
  • HTML attributes have names and values.
  • Image tags - needs a source and alt (not required but useful for when images dont render and for people that rely on readers)
  • Whitespace (br tags and p tags)
  • inline (<br>, <em> and <img>,<span>) vs block (<p>, <div>) *Block inserts formatting while in line does not

Structure of HTML document

  • Doctype
  • Open and closing html tags
  • head: stores meta data and other stuff, where you would include javascript
  • body tag: contents of your docuent

Creating a Structured Document with HTML

Tree Like Structure: Within one element you can have many. Inside a div you might have an h2 and a span within that.

Everything on a webpage is made of boxes.

You can convert boxes to HTML by starting to define boxes as div tags. You can also use class elements to help you remember what the div tags refer to. It's best practice to make sure they make sense so you can remember them later.

Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS): iss code written to control the "style" of HTML elements. It helps to avoid reppetition in your code.

Can be compared to national, state, municipal laws, and house rules(most specific). CSS is set at the national level so all other areas inherrit the rules.
For example: in CSS you have to write a selector. You can write one that applies to all paragraphs or just to a particular section of a page. You can also combine selectors.

Types of selectors:

  • ID selector: most powerful type. They beat out others when other types of selecotrs and the styles are defined within. It doesn't allow for lower-specificity.
  • ID selector: most powerful type. They beat out others when other types of selecotrs and the styles are defined within. It doesn't allow for lower-specificity.
  • Class selector: probably the most useful and versatile. Well supported by all browsers. You can add multiple classes on HTML elements.
  • Tag selector: Most useful when changing properties that are uniqe to that HTML element. For exampel setting list styles or tab-size.
  • Attribute selctor:Can be as useful as classes becuase they have the same specificity value but can be any attribute not just class.
  • Positional selectors: There are several types beyond :nth-child. You can select elements based on their position in HTML
  • Psudo selectors: :empty is one of many pseudo selectors, which you can recognize by the colon (:) in them. They typically represent something that you couldn't know by just the element and attributes alone.

How to include CSS styling on a web page:

  • Method 1: write CSS in the <head> of your HTML. Advantages: its good for small projects.
  • Method 2: Link HTML to a seperate CSS file. Extra work up front to but allows you to stay more organized when working on larger projects. You also need to link the file in your HTML file.
  • Method 3: Wite styles inline with HTML: Generally a bad idea because of a lot of repetition in your code.
